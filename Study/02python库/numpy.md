# numpy

### 数组形状

```python
a = np.array([[3,4,5,6,7,8],[4,5,6,7,8,9]])

# 查看数组形状
# (2, 6) 行， 列
a.shape

# 修改数组的形状
# 是有返回值的，需要变量接收
a.reshape(3,4)

# 将二维数组转换成一维数组
a.reshape((a.shape[0] * a.shape[1],))

# 将多维数组转换成一维数组
# reshape方法可以传递元祖也可以不使用元祖
a.reshape(1, 12);
a.flatten()

```

### 数组计算

```python
# 数组和单个数字进行计算

ndarray = [[1,2,3], [4,5,6]]
# 将每个数组做计算加减乘除
ndarray + 2

# 数组和数组之间计算
# 当数组形状相同时对应位置的加减乘除计算
# 如果两个数组形状不相同，并且有行或列的（某一维度上）形状相同，则对该行或列进行计算；反之，没有行或列的形状相同，则计算失败，程序报错。
```
